# File-Organizer-Script
A small script that I wrote awhile ago and recently started adding to again.  Planning on adding a ton of support for lots of file types.

Currently supported file types:

Audio:
-AIFF
-FLAC
-MP3
-WAVE

Documents:
-DOC
-DOCX
-HTML
-PDF
-RTF
-TXT

Images:
-GIF
-JPEG
-JPG
-Pixelmator
-PNG

Videos:
-AVI
-FLV
-MP4
-MPEG
-MOV
-WMV

Zip
